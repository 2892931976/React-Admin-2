# React-Admin
React 后台管理系统模板，React + React-Router + Webpack + ES6 + Antd , 未集成 redux， 后面会加一个集成redux的模板

## 零、计划实现
- [ ] 登录逻辑
- [ ] 表格的新增，编辑和删除
- [x] Markdown 格式文本的展示

## 一、如何运行
```bash
# 1. 下载源码
git clone https://github.com/zhongxia245/React-Admin.git

# 2. 安装依赖
npm install

# 3. 运行
npm start

# 4. 访问: http://localhost:3000
```
